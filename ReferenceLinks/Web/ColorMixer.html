ColorMixer

Summary:

      The ColorMixer is a sample lesson created as a part of a bigger project, Interactive Light Studio. The page aims to give preschool students and the teachers a template of possible mixtures of primary colors, accompanying an in class activity.

Details:

      Both the design and the functionality of this webpage is simple, as the target users for the web page are mainly preschool teachers who may not be extremely adequate in navigating through technologies.

      The web page uses two buttons to control: the "roll" button to shift through different colored shapes, and the "move" button to activate the animation that overlaps the two shapes to create the color mixing effect.
