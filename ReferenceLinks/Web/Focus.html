Focus!

Summary:

      ‘Focus!’ is a chrome extension that blocks any webpage, not including a new tab page, with a blue screen and a count-down timer that notify the user with the remaining block-out time.

Details:

      The motivation of this project is to help the target audiences to avoid distraction that comes from browsing and focus on their work at hand. The appearance of the project was designed to be very simplistic, with only a shade of blue as background and a count-down timer to denote the block out time.

      The extension gathers input through prompt answers. It will ask for the user’s permission to start the plugin, and then ask the user to customize the block out time in terms of minutes. To ensure that the user input is a valid integer input, the program will first check against possible inputs such as “15 minutes”, “minutes” and “15”, where only the last one is considered a valid input. The program will parse through the input and return only the number. Thus inputs such as “15 minutes” will be viewed the same as “15”, and inputs such as “minutes” will be returned as “undefined”. The returned values will go through a second check, which will differentiate between a number and NaN. If the input is determined to be number, then the timer will be set to that input. Else, if the input is determined to be NaN, then the timer will be set to the default value, “15”.
