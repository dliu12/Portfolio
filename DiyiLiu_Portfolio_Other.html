CuckooClock

Summary:

      CuckooClock is a Halloween themed toy clock assembly model created using Fusion360. The model won first place for SolidProfessor's 2018 CAD-loween design contest. Other than the traditional features that exist on a cuckoo clock, this model also features two wall panels that turn showing different profiles on each side. For example, one of the panels features a lollipop, and on the flip side of the same panel there is an Oni face, which is a traditional Japanese demon. The other panel features a cat on the front side and a grim reaper on the flip side.

BumbleBee

Summary:

      BumbleBee is a model designed for SolidProfessor's winter CAD design challenge using Fusion 360. The theme of the model is a children's sled, and the appearance of the model was inspired by the movie, Bumblebee. The design attempted to imitate the features of a race car by adding in wheels and tail wings. A transformer logo plate was attached to the front of the sled.
